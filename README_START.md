# 启动指南

## 启动方式

### 方式一：使用 Python 脚本（推荐）
```bash
python run.py
```

### 方式二：使用批处理文件（Windows）
双击 `run.bat` 或在命令行运行：
```bash
run.bat
```

### 方式三：直接运行主程序（开发模式）
```bash
python src/main.py
```

## 说明

- **`run.py`**：项目根目录下的主启动脚本，推荐使用此方式启动
- **`run.bat`**：Windows 批处理启动脚本，自动设置编码并启动程序
- **`src/main.py`**：原始入口点，保留用于兼容性

所有启动方式最终都会启动相同的 GUI 应用程序。
